Amazon EBS Volumes Delete on Instance Termination

By Veronique Robitaille in AWS

February 18, 2018

Did you know that by default Amazon EBS root device volumes are automatically deleted when the instance terminates? This is also the case with ephemeral storage (instance store volumes). Other non root EBS volumes are not deleted. You can configure your instance so it doesn’t delete the EBS root device volumes with the DeleteOnTermination attribute. You can find more information on this AWS EBS feature at What happens to an EBS volume when the instance is terminated?
